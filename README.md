# Link Tracker

A self-hosted link shortener with click analytics.

**Stack:** React + Vite, Express 5, PostgreSQL, Redis, Nginx, Docker Compose

**Domain:** link-tracker.doodler.dev

## Setup

Copy `.env.example` to `.env` and fill in the values.

```bash
docker compose up -d
```
